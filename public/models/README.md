# 3D Models Directory

This directory contains 3D model files for the realistic character.

## Required Models:
- human-male.glb (Realistic human character model)
- animations/ (Character animation files)

## Sources:
- Mixamo (Adobe)
- Ready Player Me
- Sketchfab
- Free3D

## Usage:
These models are loaded using @react-three/drei useGLTF hook for realistic character rendering.
